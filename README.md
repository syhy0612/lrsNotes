# 狼人杀在线笔记

狼人杀在线笔记是一个为狼人杀游戏玩家设计的工具，方便玩家记录发言信息并导出，提高游戏体验。

**项目地址**：[点击访问](https://syhy0612.github.io/lrsNotes/) 。

## 引用说明

部分图标素材取自阿里巴巴矢量图标库 [https://www.iconfont.cn/]
